# OliveHorse SEO memory

- Use only repository-backed, verified business facts. Unknown contact, price, schedule, review, qualification and imagery data is prohibited.
- Never use OAuth credentials, refresh tokens or API keys in browser code, prompts, reports, commits or logs.
- OpenRouter receives only the minimum business profile, route inventory and non-sensitive performance evidence needed for the task.
- Research and generated ideas are not publishing approval. Content requires topic approval, QC and explicit publishing approval.
- Publishing scripts default to dry-run and must never merge a pull request or publish social posts.
- Published blog content alone may enter `content/blog/` and the sitemap.
